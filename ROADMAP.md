# Roadmap

Sparal is moving toward a dependable open-source reference app for sandboxed coding agents.

## Now

- Keep the repository public and reviewable from GitHub web, profile, and API surfaces.
- Maintain a passing CI baseline for lint, TypeScript, and production build.
- Document the full local setup for Clerk, OpenAI, E2B, Inngest, Prisma, and Postgres.
- Capture fresh screenshots for the README and release notes.

## Next

- Add a local demo mode that can render the core UI without provider credentials.
- Add integration tests for project creation, message creation, and usage limits.
- Add a sandbox template publishing guide for E2B.
- Improve error reporting around agent runs and sandbox failures.
- Add security notes for generated code isolation and provider key handling.

## Later

- Support multiple sandbox templates.
- Add configurable model/provider settings.
- Add maintainer automation recipes for issue triage, docs generation, and test reproduction.
- Add observability dashboards for agent runs, tool calls, and sandbox duration.
- Publish tagged releases with migration notes.
